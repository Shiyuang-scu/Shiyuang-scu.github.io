**根为总分类**，和 Java 的 Object 类似。

分为四层(待定)：

1. 总分类(Root)
2. 领域大类(父概念只有总分类，属于二级概念)
3. 过渡类(父概念只有总分类以及领域大类，在这一层之下就没有树状结构了)
4. 本体类(父概念为2类或者3类，这一层会有各种的父子概念交叉，弱化层级结构)
5. 基底类(不再有子类的类)

每一个本体有如下元素

+ 名称
+ 描述
+ 同义词
+ 父概念
+ 子概念
+ 属性
+ 实例

用 json 描述，包括把书籍和笔记的标签
